# Framework Icons

This directory contains placeholder imagesets for framework logos. To complete the implementation, add actual icon PNG files to each imageset folder.

## Required Icons

Each icon needs three sizes:
- `{name}-icon.png` (1x - 16x16 or 32x32)
- `{name}-icon@2x.png` (2x - 32x32 or 64x64)
- `{name}-icon@3x.png` (3x - 48x48 or 96x96)

## Icon Sources

You can download official logos from:
- **Next.js**: https://nextjs.org/
- **Vite**: https://vitejs.dev/
- **React**: https://react.dev/
- **Django**: https://www.djangoproject.com/
- **Node.js**: https://nodejs.org/
- **Python**: https://www.python.org/
- **Prisma**: https://www.prisma.io/
- **Docker**: https://www.docker.com/

## Fallback Behavior

Until actual icon files are added, the app will use SF Symbols as fallbacks:
- `laptopcomputer` for unknown frameworks
- `globe` for external websites
